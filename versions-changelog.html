📝 Detailed history of every Novel DR release 📝

🔖 Version 1.3

Released: April 8, 2025
  • Added – Sepia reading theme (easy on eyes for night reading).
  • Added – Auto‑scroll speed presets (slow, medium, fast).
  • Added – Backup & restore entire library metadata (JSON export/import).
  • Added – Batch delete multiple novels from library.
  • Improved – Download stability for novels with 2000+ chapters.
  • Improved – Chapter list loading speed (virtualized).
  • Fixed – Crash when reopening app after long background time.
  • Fixed – Line spacing resetting after theme change.

🔖 Version 1.2

Released: February 18, 2025
  • Added – Font size presets (14pt, 16pt, 18pt, 20pt, 22pt).
  • Added – Line spacing options (1.0, 1.2, 1.5, 1.8).
  • Added – Download logs (view which chapters failed).
  • Improved – Reading progress indicator in chapter list.
  • Fixed – Rare issue where novels wouldn't resume download after interruption.
  • Changed – Minimum Android version raised to 8.0 (Oreo).

🔖 Version 1.1

Released: January 5, 2025
  • Added – Update novels: fetch only new chapters (no re-download).
  • Added – Table of contents modal with current chapter highlight.
  • Added – Reading progress saved per chapter (scroll position).
  • Added – Haptic feedback toggle (optional).
  • Improved – Dark theme contrast.
  • Fixed – Bug where some novels failed to parse due to site changes.

🔖 Version 1.0

Released: December 12, 2024
  • Initial stable release.
  • Download entire novels from ReadNovelFull, NovelFull, FreeWebNovel.
  • Offline reader with customisable font size (basic).
  • Two themes: Light and Dark.
  • Library management: delete, reorder, mark read.
  • Basic auto‑scroll (fixed speed).
  • Persistent storage using AsyncStorage.

🔖 Version 0.9 Beta

Released: November 20, 2024
  • First public beta.
  • Core download and reading functionality.
  • Experimental support for three novel sites.
  • Basic library with local storage.
  • Known issues: occasional crashes on very long chapters, no update feature.
  • For testing only – not recommended for daily use.
