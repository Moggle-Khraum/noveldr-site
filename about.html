📖 About this site

This is the official homepage of Novel DR – a free, open‑source Android app for downloading and reading Wuxia, Xianxia, and light novels offline. The site is designed with a nostalgic 2005–2012 web aesthetic (gradients, rounded corners, old‑school buttons) to celebrate the golden era of personal blogs and early indie software.

The website provides the latest APK, a full feature list, an archive of old versions, visitor counters, and links to the source code. No trackers, no cookies – just pure HTML/CSS/JS.

📱 About the App (Novel DR)

Novel DR is an offline novel reader for Android. You can download complete novels from three supported websites (ReadNovelFull, NovelFull, FreeWebNovel) and read them anywhere without an internet connection.

  • ✔ Download entire novels chapter by chapter
  • ✔ Customizable font size (14–22pt) and line spacing
  • ✔ Three themes: Dark, Light, Sepia
  • ✔ Auto‑scroll mode with adjustable speed
  • ✔ Reading progress saved per chapter (scroll position)
  • ✔ Table of contents with current chapter highlight
  • ✔ Backup & restore library metadata (JSON)
  • ✔ Update novels – fetch only new chapters
  • ✔ 100% free, no ads, no tracking

The app is built with React Native (Expo) and targets Android 8.0+. All source code is available on GitHub.
